# RAG Test

## Installation

  Fill in Open API key

  ```bash
  $ cp .env.sample .env
  # then fill in .env
  ```

- Install dependencies

  ```bash
  # uv(default)
  $ uv install
  
  # pdm
  $ pdm install
  ```

## Run

- jupyterlab

  ```bash
  $ uv run jupyter lab
  ```

- command line

  ```bash
  $ uv run python3 main.py
  ```
